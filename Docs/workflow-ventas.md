# Flujo de trabajo: Ventas en tienda

Este documento describe el flujo de trabajo principal para las ventas en mostrador.

## Flujo de ventas en tienda

1. El cliente llega y solicita uno o varios productos.
2. El trabajador registra al cliente si es nuevo o selecciona uno existente (o un cliente genérico de mostrador si aplica).
3. El trabajador busca los productos en el sistema, verifica el stock disponible y arma la venta con las cantidades solicitadas.
4. El sistema calcula subtotal, descuentos, impuestos y total; el trabajador revisa y confirma el monto con el cliente.  
5. El cliente elige el método de pago (o una combinación de métodos) y el trabajador registra los pagos hasta cubrir el total.
6. El sistema registra la venta, descuenta el stock de cada producto vendido y genera el comprobante (ticket o factura) para el cliente.
