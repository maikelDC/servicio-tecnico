# Flujo de trabajo: Servicio técnico

Este documento describe el flujo de trabajo principal para el ingreso y reparación de equipos en el taller.

## Flujo de ingreso y reparación

1. El cliente llega con un equipo para revisión o reparación.
2. El trabajador registra o selecciona al cliente y registra o selecciona el dispositivo asociado al cliente.
3. Se crea una orden de servicio con el dispositivo, el problema reportado, el estado inicial, el usuario que recibe y, cuando aplica, fotos de entrada.
4. El técnico revisa el equipo, realiza el diagnóstico y registra en la orden el diagnóstico, el trabajo estimado y los posibles repuestos necesarios.
5. El trabajador comunica al cliente el presupuesto (mano de obra y repuestos) y se registra si el cliente aprueba o rechaza la reparación, actualizando el estado de la orden.
6. Si la reparación es aprobada, el técnico realiza el trabajo, registra el trabajo realizado, los repuestos realmente utilizados y, cuando corresponde, las fotos de salida.
7. Al finalizar la reparación, la orden pasa a un estado de lista para entrega y se notifica al cliente que el equipo está listo para retirar.
8. Cuando el cliente llega a retirar el equipo, se revisa junto al cliente, se cambia la orden a estado entregada y se registra el usuario que realiza la entrega.
9. Se genera la venta asociada a la orden de servicio (incluyendo mano de obra y repuestos utilizados), se registran los pagos correspondientes y se descuenta del inventario cada repuesto utilizado.
